# 1DAL-WEB PRD (Product Requirements Document)

## 📌 제품 개요
`onedal-web`은 `onedal-app`이 던져주는 **콜 데이터를 실시간으로 중계받고**, 카카오내비 연동, 운임 확인, 그리고 수익성 판단을 돕는 **"웹 기반 운전석 대시보드 및 지휘 통제소"** 입니다. 

## 🎯 핵심 목표
1. **실시간성 보장 (Socket.io)**: 안드로이드 스캐너 폰(1호기)에서 받은 콜 정보를 딜레이 없이 즉시 관제 폰(2호기) 화면에 띄웁니다.
2. **휴먼 인 더 루프 (Human-in-the-loop)**: 스캐너가 임시 선점(상세페이지 진입) 해둔 콜의 출/도착지, 운임을 보고 운전자가 최종 수락을 결정할 수 있도록 시각화 및 전화 딥링크를 지원합니다.
3. **무중단 운영 (Wake Lock)**: 운전 내내 브라우저 화면이 꺼지지 않아야 합니다.

## 🔥 주요 기능 요구사항

### 1. 실시간 대시보드 (`/`)
- 카카오내비처럼 화면이 절대 꺼지지 않도록 Wake Lock API 활성화.
- Socket.io 연결 상태 모니터링 (연결됨/끊김 상태 표시).
- 콜 수신 시 시인성 높은 큰 카드 레이아웃으로 데이터 표시 및 **알림음(꿀콜 사운드)** 재생.
- 카드 내비게이션 버튼: `[📞 상차지 전화]` (tel 딥링크), `[🗺️ 카카오내비]` (kakaonavi 딥링크) 버튼 클릭 가능.

### 2. 정산 및 통계 페이지 (`/settlement`)
- 오늘/이번 주/이번 달 누적 콜 수 및 확정된 콜의 누적 수익을 요약하여 시각화.
- 확정 건에 대해 완료 처리 기능 제공.

### 3. 카카오 노선 계산기 (추후 확장)
- 카카오 모빌리티 API를 프록시 통신하여, 기존 진행 중인 콜 대비 새로 추가된 콜의 '거리 편차(우회율)와 추가 소요 시간'을 자동 연산하여 대시보드 콜 카드에 띄워줍니다.
