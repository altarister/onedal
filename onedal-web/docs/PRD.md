# 1DAL-WEB PRD (Product Requirements Document)

## 📌 제품 개요
`onedal-web`은 `onedal-app`이 던져주는 **콜 데이터를 실시간으로 중계받고**, 카카오내비 연동, 운임 확인, 그리고 수익성 판단을 돕는 **"웹 기반 운전석 대시보드 및 지휘 통제소"** 입니다. 
기존에는 폴링(Polling) 방식의 Next.js 라우터를 사용했으나, **100% 무손실 실시간(Socket.io)** 데이터 전송을 위해 독립된 프론트/백엔드 체제로 재구축되었습니다.

## 🎯 핵심 목표
1. **실시간성 보장 (Socket.io)**: 안드로이드 스캐너 폰(1호기)에서 받은 콜 정보를 딜레이 없이 즉시 관제 폰(2호기) 화면에 띄웁니다.
2. **휴먼 인 더 루프 (Human-in-the-loop)**: 스캐너가 임시 선점(상세페이지 진입) 해둔 콜의 출/도착지, 운임을 보고 운전자가 최종 수락을 결정할 수 있도록 시각화 및 전화 딥링크를 지원합니다.
3. **무중단 운영 (Wake Lock)**: 운전 내내 브라우저 화면이 꺼지지 않아야 합니다.

## 🔥 상세 기능 및 페이지 요구사항

### 1. 실시간 대시보드 (`client/src/App.tsx`, `/`)
- **Wake Lock**: `navigator.wakeLock.request('screen')`을 적용하여 카카오내비처럼 화면 꺼짐 방지.
- **연결 모니터링**: Socket.io 연결됨/끊김 상태를 우측 상단에 🟢/🔴 점과 텍스트로 표시.
- **콜 카드(오더 카드)**: 
  - 수신 즉시 역순(최신순) 정렬되어 렌더링.
  - 출발지, 도착지, 요금을 큰 글씨 타일로 표시.
- **오디오 피드백**: 새 오더 수신 시 Web Audio API 코드로 "띵띵(880Hz -> 1100Hz)" 시스템 경고음 강제 재생.
- **딥링크 액션 버튼**: 
  - `[📞 상차지 전화]`: `href="tel:번호"` 터치 시 즉시 전화 앱 팝업.
  - `[🗺️ 카카오내비]`: `kakaonavi://navigate` 스키마 응용.

### 2. 정산 및 통계 페이지 (`client/src/pages/Settlement.tsx`, `/settlement`)
- 오늘 콜 수 렌더링.
- 최종 확정 건수에 따른 총 운송 수익금액 집계.
- 일간 콜 내역 데이터 리스트업 및 '확정' 마킹 지원.
