# 1DAL 글로벌 기획서 (Global PRD)

본 문서는 `onedal-app`과 `onedal-web` 두 개로 나뉘어진 1DAL 시스템이 어떻게 하나의 거대한 서비스로 융합되어 수익을 창출하는지를 정의합니다. 

## 🔗 제품 상관관계 (Product Ecosystem)

1DAL 시스템은 **기계의 압도적인 속도**와 **인간의 신중한 판단력**을 결합한 '반자동 칵핏(Cockpit)' 구조입니다.

- **onedal-app (척후병 역할)**: 
  - 보이지 않는 글로브박스나 에어컨 송풍구 쪽에 24시간 방치해두는 안드로이드 공기계(스냅드래곤 8, RAM 6GB 이상 권장) 전용 앱입니다. 
  - 0.1초의 레이턴시로 남들이 보기 전에 콜을 잡아두거나(상세 페이지 진입 대기), 버려지는 데이터를 모읍니다. 사용자 간섭률 0%.
- **onedal-web (사령관 역할)**: 
  - 운전자 시야 정면 거치대에 두는 업무 전용 스마트폰(알뜰폰 주력회선 장착)의 크롬 브라우저 화면입니다.
  - 척후병이 던져준 1차 선별 데이터 중, 진짜 '돈이 되는' 콜인지 거리와 유류비(우회율)를 계산합니다.
  - 내비 폰에 시각적인 카드와 소리로 알림을 울려주어 사령관(운전자)이 화주에게 전화를 걸고 30초 내에 취소/확정을 최종 보스처럼 결정하게 돕습니다. Wake Lock 화면 무한 유지 기능으로 무중단 운영됩니다.

- **안드로이드 기획 상세**: `onedal-app/docs/PRD.md`
- **웹 대시보드 기획 상세**: `onedal-web/docs/PRD.md`
