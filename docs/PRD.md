# 1DAL 글로벌 기획서 (Global PRD)

본 문서는 `onedal-app`과 `onedal-web` 두 개로 나뉘어진 1DAL 시스템이 어떻게 하나의 거대한 서비스로 융합되어 수익을 창출하는지를 정의합니다. 상세 기획은 각 폴더 내부의 문서를 참고하세요.

## 🔗 제품 상관관계 (Product Ecosystem)

1DAL 시스템은 **기계의 속도**와 **인간의 판단력**의 장점을 결합한 '반자동 칵핏(Cockpit)' 구조입니다.

- **onedal-app (척후병 역할)**: 
  - 앱스토어 배포용이 아닌, 오직 승욱님의 공기계에만 수동으로 설치되는 해킹에 가까운 스캐너입니다. 
  - 0.1초의 레이턴시로 남들이 보기 전에 콜을 잡아두거나(상세 페이지 진입 대기), 버려지는 데이터를 모읍니다.
- **onedal-web (사령관 역할)**: 
  - 척후병이 던져준 1차 선별 데이터 중, 진짜 '돈이 되는' 콜인지 거리와 유류비를 계산합니다.
  - 내비 폰에 시각적인 카드와 소리로 알림을 울려주어 사령관(운전자)이 화주에게 전화를 걸고 30초 내에 취소/확정을 최종 보스처럼 결정하게 돕습니다.
