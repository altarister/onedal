# 1DAL-APP PRD (Product Requirements Document)

## 📌 제품 개요
`onedal-app`은 인성앱 화면의 **콜(오더) 정보를 초고속으로 실시간 스캔**하고, 설정된 기준에 맞는 꿀콜을 **0.1초 만에 자동 선점(클릭)**하는 안드로이드 네이티브(Kotlin) 자동화 엔진입니다.

## 🎯 핵심 목표
1. **0.1초 선점**: 사람이 볼 틈도 없이 기계적인 속도로 조건에 맞는 콜의 상세 페이지 진입(터치).
2. **휴먼 인 더 루프(Human-in-the-loop) 대비**: 배차 '확정' 버튼은 직접 누르지 않고, 상세페이지 진입까지만 하여 30초 대기 시간을 멉니다.
3. **빅데이터 축적 (Intel)**: 버려지는 똥콜 정보들도 폐기하지 않고 백그라운드에서 수집하여 서버로 쏴줍니다(HTTP POST).

## 🔥 주요 기능 요구사항

### 1. 화면 스캔 및 노드 추출
- 스마트폰 화면(인성앱)에 뜨는 콜 리스트 텍스트를 인식합니다.
- `상차지`, `하차지`, `요금` 세 가지 필수 데이터를 추출합니다.

### 2. 조건 필터링 (수익성 판별)
- 앱 내에 설정해 둔 '기본 단가', '특정 선호 지역(ex. 성남/광주)'에 부합하는지 즉시 계산합니다.
- **합격 콜**: 즉시 자동 터치 수행.
- **불합격 콜**: 리스트업 후 백그라운드에서 서버로 조용히 전송 (인텔 데이터).

### 3. 무결점 자동 터치 (Auto-Click)
- 합격 콜 판별 시 딜레이 없이 해당 텍스트/버튼 노드의 X, Y 좌표를 계산하여 시스템 레벨 터치 이벤트를 발생시킵니다.

### 4. 오버레이 UI (상태창)
- 인성앱 위에 조그맣게 '1DAL 엔진 가동 중 상태값(초록/빨강)'을 띄울 수 있는 플로팅 위젯을 제공합니다.
